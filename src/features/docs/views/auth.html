Authentication

Spider provides a complete authentication system with JWT, cookies, and Google OAuth integration.

JWT Authentication

Sign and verify JSON Web Tokens with built-in functions:

zig
const auth = spider.auth;

// Sign a JWT token
const claims = .{
.sub = user_id,
.email = user_email,
.exp = std.time.timestamp() + 3600, // 1 hour expiration
};
const token = try auth.jwtSign(allocator, claims, jwt_secret);
defer allocator.free(token);

// Verify a JWT token
const verified_claims = try auth.jwtVerify(allocator, token, jwt_secret);
defer {
allocator.free(verified_claims.email);
}

Cookie Management

Handle cookies for authentication sessions:

zig
// Set authentication cookie
const cookie_value = try auth.cookieSet(allocator, token);
defer allocator.free(cookie_value);
try res.headers.set(allocator, "Set-Cookie", cookie_value);

// Get token from cookie
const cookie_header = req.headers.get("Cookie") orelse return error.Unauthorized;
const token = auth.cookieGet(cookie_header) orelse return error.Unauthorized;

// Clear authentication cookie
const cleared_cookie = try auth.cookieClear(allocator);
defer allocator.free(cleared_cookie);

Google OAuth

Integrate Google authentication with built-in support:

zig
const google = spider.google;

// Generate Google OAuth URL
const config = google.GoogleConfig{
.client_id = "your-client-id",
.client_secret = "your-client-secret",
.redirect_uri = "http://localhost:8080/auth/google/callback",
};
const auth_url = try google.authUrl(allocator, config);
defer allocator.free(auth_url);

// Handle callback and get user profile
const profile = google.GoogleProfile{
.id = "google-user-id",
.email = "user@gmail.com",
.name = "User Name",
.picture = "https://...",
};
